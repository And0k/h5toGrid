SMS2GPX: Convert XML file of backuped SMS messages received from GPS tracker to GPX tracks
version= 0.0.1
© 2016 Andrey Korzh <korzh@nextmail.ru>

 Позволяет получить из XML файла программы "SMS Backup & Restore" данные
трекеров в формате GPX (которые можно открывать в программах "Garmin
Mapsource", "OpenCPN", "GPX Editor", "qGIS" и многих других)

Конфигурация задается файлом SMS2GPX.ini
